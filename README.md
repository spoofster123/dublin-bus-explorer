# Dublin Bus Explorer

A Scandinavian‑designed, frontend‑only transit analysis tool.

## Features
- Live Dublin Bus vehicle positions
- Route connectivity explorer (1–3 hops)
- Scandinavian UI (light + dark mode)
- Floating legend
- Route details drawer
- 100% client‑side — no backend

## Deployment
Enable GitHub Pages:
Settings → Pages → Deploy from branch → main → root
